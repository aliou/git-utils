git-wip(1) -- Quickly make WIP commits
=============================================

## SYNOPSIS

`git wip` [-m MESSAGE] [FILES]

## DESCRIPTION

**git-wip** creates a git commit with prefixed by "[WIP]". If given, it only
commits the file in the argument list.

## OPTIONS

  * `-m`
    Specify a message.

## SEE ALSO

git(1), git-add(1), git-commit(1)
